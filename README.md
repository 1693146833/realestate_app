# realestate_app
共有三个页面，分别是首页，主页面和描述页面。想体验内容可右侧的release框下载，该app用的是java在Androidstudio开发。启动app首先显示首页，之后点击按钮显示主页面，可点击房屋图片，即可进入描述页面。
